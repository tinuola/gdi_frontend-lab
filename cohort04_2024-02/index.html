What & Why?

Cohort Goals

  • Building projects for our portfolios
  • Using and improving our Frontend skills
  • Cultivating a consistent coding habit
  • Feedback about coding best practices
  • Learning from each other

Cohort Pre-requisite

Some experience with HTML, CSS and JavaScript

Cohort Structure

  • A new coding challenge every week
  • Weekly lab sessions (1.5 - 2 hours)
  • Code review and feedback during lab
  • Private Slack channel for ongoing support
